3.12 Calling Procedures

Procedure Definition A named segment of code that can be called and executed elsewhere.

Parameters, which are input values, can be defined in a procedure. Arguments are values inputted into a procedure call. Values can be returned from a function.

Pseudocode Reference

Hover over each item below to learn more.

  • PROCEDURE procedureName(parameter1, ...) {}Define a procedure
  • procedureCall(argument1, ...)Call a procedure
  • RETURN(expression)Return a value from inside a procedure

Premade Pseudocode Procedures

Hover over each item below to learn more.

  • DISPLAY(expression)Accept a value from the user and returns the value
  • INPUT()Display a value to the user
